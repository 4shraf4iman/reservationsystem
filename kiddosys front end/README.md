
# kiddosys

## Requirements

### Node Modules

1. Clone the repository:
   ```bash
   git clone https://github.com/4shraf4iman/kiddosys.git && cd kiddosys
   ```

2. Install the required Node modules:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Set up the localhost in the `services.ts` file.
